ตัวแปร และชนิดข้อมูล

ตัวแปร (Variable) เปรียบเสมือน "กล่อง" ที่เราแปะป้ายชื่อเอาไว้ใส่ข้อมูล ข้อมูลมีหลายประเภท เช่น ตัวเลข (Integer) หรือ ข้อความ (String)

# ตัวอย่างการสร้างตัวแปร
name = "สมชาย" # String (ข้อความ)
age = 14 # Integer (จำนวนเต็ม)
grade = 3.50 # Float (ทศนิยม)
เข้าสู่บทเรียนที่ 2 (ดูวิดีโอ & ทำแบบฝึกหัด)
บทที่ 3

รหัสควบคุม & ตัวดำเนินการ

การคำนวณใน Python ใช้เครื่องหมายทางคณิตศาสตร์ และมีรหัสพิเศษบางตัวที่ควรรู้

+ - * / : บวก ลบ คูณ หารทั่วไป
** : ยกกำลัง (เช่น 2**3 = 8)
// : หารปัดเศษทิ้ง (7 // 2 = 3)
% : หารเอาเศษ (7 % 2 = 1)
เข้าสู่บทเรียนที่ 3 (ดูวิดีโอ & ทำแบบทดสอบ)

ฟังก์ชัน print และ input

หัวใจของการโต้ตอบ

print() ใช้แสดงผลข้อมูลออกทางหน้าจอ
input() ใช้รับข้อมูลจากแป้นพิมพ์ (ได้เป็นข้อความเสมอ)
name = input("คุณชื่ออะไร: ")
print("สวัสดี", name)

โครงสร้างแบบเรียงลำดับ

โปรแกรมปกติจะทำงานจาก บรรทัดบนสุด ลงมา ล่างสุด ทีละบรรทัด เราต้องเรียงลำดับคำสั่งให้ถูกต้อง เช่น ต้องรับค่าก่อน แล้วค่อยนำมาคำนวณ

โครงสร้างแบบเลือกทำ (if-else)

ทำให้โปรแกรม "คิด" เองได้

เราใช้คำสั่ง if เพื่อตรวจสอบเงื่อนไข ถ้าจริงทำอย่างหนึ่ง ถ้าไม่จริงทำอีกอย่าง

score = 50
if score >= 50:
    print("สอบผ่าน")
else:
    print("สอบตก")
